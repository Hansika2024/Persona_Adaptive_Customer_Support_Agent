# Login Troubleshooting Guide

## Overview

This guide helps resolve common login issues experienced by users.

---

## Common Login Problems

### Invalid Credentials

Users may receive an error when entering incorrect email or password.

### Account Locked

Accounts may be temporarily locked after multiple failed login attempts.

### OTP Verification Failure

Multi-factor authentication may fail if OTP expires.

---

## Troubleshooting Steps

### Step 1: Verify Credentials

Ensure:

* Correct email address
* Correct password
* Caps Lock is off

### Step 2: Reset Password

If password is forgotten:

1. Click "Forgot Password"
2. Enter registered email
3. Follow reset instructions

### Step 3: Clear Browser Cache

Cached browser data may cause login issues.

Recommended:

* Clear cache
* Restart browser

### Step 4: Check Internet Connectivity

Ensure stable internet connection.

---

## Additional Support

If login issues continue, contact support with:

* Email address
* Screenshot of error
* Browser details
